🔌 API Endpoints

Avatar SVG API

Get avatar as SVG (universal mode with all 20 states).

/avatar/{input}.svg

?frame=happy - Set initial state
?legacy=true - Legacy single-frame mode

Avatar PNG API

Get avatar as PNG image (requires frame parameter).

/avatar/{input}.png?frame={state}

Avatar Info API

Get detailed information about an avatar's generation parameters.

/avatar/{input}.svg/info

Animation Frames API

Get metadata about all 20 available animation frames.

/avatar/{input}.svg/frames

Avatar Bundle API

Get all frames as a ZIP bundle for download.

/avatar/{input}/bundle

Version Info API

Get current version and feature information.

/version

Health Check API

Simple health check endpoint.

/health

Debug Info API

Detailed debug information for an avatar (development only).

/debug/avatar/{input}

🎨 Available Animation States

Emotes (5): neutral, happy, sad, surprised, angry, bored
Idle Frames (10): idle_0 through idle_9 - Independent eye/mouth combinations for natural idle animation
Vowels (5): vowel_a, vowel_e, vowel_i, vowel_o, vowel_u - For lip-sync animation
