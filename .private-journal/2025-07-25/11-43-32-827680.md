## Project Notes

Successfully documented the new hair asset positioning system in both README.md and CLAUDE.md. The documentation covers:

1. Updated asset structure to include hair/ directory
2. Comprehensive explanation of all SVG data attributes:
   - data-offset-y for fine positioning
   - data-z-order for render order (behind/front)
   - data-width-percent for sizing
   - data-position-x for horizontal alignment (body-center/cell-center)
   - data-position-y for vertical positioning (above-body/between-body-eyes)
3. Color inheritance using currentColor
4. Example SVG markup showing proper usage

The system is now fully documented and ready for use by future developers or users wanting to create custom hair assets. The data attribute approach makes the positioning logic self-contained within each SVG file rather than hardcoded in the script.
