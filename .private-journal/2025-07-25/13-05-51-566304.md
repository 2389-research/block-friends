## Project Notes

Found comprehensive data attribute system in sprite generator - hair assets use 6 different data attributes for positioning, sizing, z-order, and coloring. System is quite sophisticated with percentage positioning, multiple anchor modes, and flexible color specifications including JSON arrays. The parse_defs function is the key entry point that extracts all these attributes from SVG files.
