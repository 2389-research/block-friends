## Project Notes

Added CSS sprite sheet generation to the door agent generator. Creates a complete CSS file with a base .sprite class that handles sizing and background image setup, plus individual .agent-{id} classes for each of the 1500 agents with proper background-position values. This makes it incredibly easy to use specific agents in web projects - just apply both classes to an element. The CSS includes helpful comments with sheet dimensions and cell size info.
